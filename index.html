BlockGraph

BlockGraph is a Unity project which uses elements of Unity's Universal Render Pipeline (URP) and ShaderGraph system for defining shaders. It creates fun mathematical animated shapes with any number of blocks which are colored by the shader code, based on their XYZ coordinates. Effort was put into optimizing the render pipeline for minimum frame render time.
Adapted from Jasper Flick's tutorial on Catlike Coding.
